OpenRisk is an open‑source desktop application that consolidates disparate risk‑intelligence data into a single, locally‑hosted platform.  It helps businesses, not‑for‑profit organisations and compliance professionals perform Anti‑Money‑Laundering (AML), Know‑Your‑Customer (KYC), Environmental‑Social‑Governance (ESG) and other due‑diligence checks.  The tool integrates various (open) data sources such as **OpenSanctions**, **OpenCorporates** and **ADVERSEA** into an extensible plug‑in architecture.  Unlike commercial cloud‑based solutions, OpenRisk processes all data on the user’s device, which ensures **full compliance with the Digital Operational Resilience Act (DORA)** and preserves data sovereignty.  Users can combine sanctions lists, beneficial‑ownership records and adverse media signals to calculate risk scores without exposing sensitive information to third parties.
